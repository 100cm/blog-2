2012-06-17 22:47:29   查看原文>>
PostgreSQL 2012 中国用户大会于6月15号到17号在中国人民大学成功举办完成.
本次大会聚集了国内从事PostgreSQL相关工作或在工作中使用PostgreSQL的技术专家（去哪儿,阿里巴巴,腾讯,中科院,广州正虹,杭州斯凯,EMC,VMWARE等众多公司的技术专家做了相关的主题分享.)
以及高校的对PostgreSQL有深入研究和丰富教学和实践经验的中国人民大学的张孝老师，武汉大学的彭煜伟老师。
同时本次大会还邀请到了英国, 俄罗斯, 瑞典, 日本的simon, andrew, magnus,  Koichi Suzuki 嘉宾, PostgreSQL核心组员以及PostgreSQL-XC的架构师.
大会的内容简介详见:
http://wiki.postgresql.org/wiki/Pgconfchina2012 
PPT可以到http://wenku.it168.com下载，搜索关键词postgresql2012.
或者到PG社区BBS下载 : 
http://bbs.pgsqldb.com/client/post_show.php?zt_auto_bh=55858

以下是大会中的一些掠影, 由于本人拍照水平有限, 有些主题听的太投入忘记拍照了, 对这些嘉宾表示抱歉, 等照片补齐之后会陆续补上.
在人民大学逸夫馆拍的集体照 : 
 
1. 用户会会长, 李元佳(Galy)正在对本次大会致辞.

 
2. Magnus Hagander讲解9.2的新特性和路线图
其中有些特性非常有趣, 也拍了几张仅供分享.
 
 
 
 
 
 
 

3. 日本的Koichi Suzuki为我们分享了PostgreSQL 在日本电信(NTT)中的应用情况以及他们在PostgreSQL代码方面的贡献, PostgreSQL-XC等.
 
 
 
 

 
 
4. andews 是俄罗斯FREEBSD社区的核心成员之一, 听说我们在搞PG大会, 自费来参加，了解中国PG的使用情况. 真的很有心, 他们目前有3PB的raw数据量，1500W的媒体对象, 类似在线多媒体的业务。后台使用的是PostgreSQL数据库. 
 
 
5. 来自英国的simon(PostgreSQL的核心组员之一)为我们分享了企业级PostgreSQL流复制的未来以及流复制未来的发展方向.
 
 
 
 
 

6. 何伟平前辈的照片我光顾听了, 没有拍到, 等拿到照片再贴上来.
 
第二天的DBA和内核专场 : 
1. 去哪儿的罗翼为我们分享的DataBase As Application
 
 

2. 斯凯网络的谭峰正在分享Oracle VS PG的主题

3. 斯凯网络的德哥(Digoal.zhou)正在分享PostgreSQL 经验谈的主题, 图未拍, 后面再释出.

4. 阿里巴巴的唐成为我们带来了GreenPlum的经验谈.
 
 
 4. 人民大学的张孝老师给我们介绍高校中数据库教学中PostgreSQL的学习、使用与定制简介
  
 
 
 
 
 
 

5. 来自腾讯的肖涛为我们分享的是数据库压力如何向应用层分摊的主题, 肖大侠以前是我在斯凯的同事, 有很丰富的C的编程经验.
 
 
6. 武汉大学的彭煜伟老师给我们介绍 PostgreSQL 内核简介 , 彭煜伟老师是PostgreSQL内核分析一书的作者之一. 对PostgreSQL内核非常了解.
 

7. 泥鳅老师来了,分享的是Slony-II复制.
 
 
8. Gavin Yang(VMWARE), 为我们分享的PostgreSQL的数据分析工具MADlib. 非常强大.有兴趣的同学也可以为开源献出一份力量.
 
 

还有几位专家的照片还没有整理出来, 继续等待发布.
