CentOS 6.x partx instead partprobe refresh modified partition table

2014-05-04 16:03:51   查看原文>>
在CentOS 5.x中如果使用fdisk删除, 修改或创建了新的分区, 使用partprobe devname即可刷新分区表.
但是到了CentOS 6.x的版本, partprobe不管用了.
需要使用partx
例如
partx -v -a /dev/sdb
