windows7下虚拟AP共享上网

2014-07-10 15:05:25   查看原文>>
1. 首先要确保无线网卡驱动支持AP
以管理员身份运行 cmd.exe
执行命令 : 
netsh wlan show drivers
    支持的承载网络  : 是
表示支持

2. 然后要确保已经开启无线, 没有开启的话找一下笔记本的Fn功能键+无线塔的按钮开启它.

3. 接下来创建AP, 执行命令
netsh wlan set hostednetwork ssid=wifi key=teS()2_32*few mode=allow keyUsage=persistent

4. 启动AP, 执行命令
netsh wlan start hostednetwork

5. 共享本地连接给刚创建的无线AP连接.

6. 设置, 查看无线AP连接的状态.
只留下IPv4即可, 其他的都可以不勾选.

记住这个IP.


7. 使用手机或平板或其他无线设备连接这个AP.
最好手工配置平板的网络, 例如 : 
WPA2
SSID wifi
密码 teS()2_32*few
IP 192.168.137.2/24
网关 192.168.137.1
DNS 8.8.8.8, 202.101.172.35

连接变成这样就OK了

8. 其他, 可以去掉 "允许计算机关闭此设备以节约电源" 的选项 .
