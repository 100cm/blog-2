conntrack tools (TCP SESSION table SYNC over Hosts) install

2015-01-30 12:44:42   查看原文>>
conntrack tools可以用来在多主机间同步TCP会话, 时间TCP会话在超时时间内的不丢失漂移,(结合VIP漂移). 对应用程序透明.

测试环境CentOS 6.6 x64

2.6.32-504.8.1.el6.x86_64
