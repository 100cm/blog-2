compare Array's elements's class when the Array object created by %w,%W,or []

2012-03-01 12:28:11   查看原文>>
看个例子 : 
a1 = [1,"ab",1.1]
a2 = %w/1 ab 1.1/
a3 = %W/1 ab 1.1/
a1.each {
  |x|
  print("#{x.class} ")
}
a2.each {
  |x|
  print("#{x.class} ")
}
a3.each {
  |x|
  print("#{x.class} ")
}
执行结果 : 
Fixnum String Float String String String String String String 
使用%w或%W 创建的都是String类型.
使用[] 则是创建时的类型.
